# Ejemplo_RMD
Repositorio con un ejemplo del uso de RProject y RMD

## RProject
Este tipo de archivo te permite establecer el working directory (wd) en el lugar en donde se encuentra el archivo

Tienes que abrir el archivo que termina en `.Rproj`, esto abrirá tu entorno de trabajo dentro del proyecto, después de ello ya podrás abrir cada archivo RMD o R que tengas

## main.rmd
Este es el archivo principal para unir todos tus RMD, es el que debes de copilar siempre para tener el archivo final, además debes de incluir sobre este archivo las paqueterías que estas usando

## Ejerciciok.rmd
Estos archivos tienen el contenido de cada ejercicio o contenido que quieras agregar a tu RMD final para cada sección, de preferencia mantenlos en la ruta `./Ejercicios/EjercicioK.rmd`

## Auxiliares
Para poder usar paqueterías especiales u opciones de estilo personalizadas en tu archivo pdf final en la carpeta `./Auxiliares` se encuentran 2 archivos:

### preambulo.tex
En este archivo puedes agregar opciones de estilo para tu documento final, como por ejemplo el tipo de letra, el tamaño de letra, el interlineado, etc.

### portada.tex
En este archivo puedes agregar una portada personalizada para tu documento final, puedes agregar el título, el autor, la fecha, etc.
